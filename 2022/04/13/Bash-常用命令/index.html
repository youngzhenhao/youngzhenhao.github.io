Bash 常用命令

一些 Linux 常用的 bash 命令。

基本操作

export

显示所有的环境变量

获取某变量详细信息，echo $VARIABLE_NAME

whereis

使用系统自动构建的数据库搜索可执行文件，源文件和手册页面
whereis name

which

在环境变量PATH指定目录搜索可执行文件
此命令将打印可执行文件的完整路径
which program_name

clear

清除窗口上的内容

文件操作

ls

列出文件
ls option

-l详细信息
-t按时间排序
-a列出所有文件，包括隐藏文件

touch

创建或更新文件
touch filename

cat

在屏幕上显示文本文件
复制文本文件
合并文本文件
创建新的文本文件
cat filename
cat file1 file2
cat file1 file2 > newcombinedfile

more

显示文件的第一部分
(用空格移动并键入q以退出）
more filename

head

输出文件的前10行
head filename

tail

输出最后10行文件
-f在文件增长时输出附加数据
tail filename

mv

将文件从一个位置移动到另一个位置
mv filename1 filename2

filename1文件源路径
filename2目标路径

cp

将文件从一个位置复制到另一个位置
cp filename1 filename2

filename1文件源路径
filename2目标路径

rm

删除文件
rm filename
添加-rf递归删除目录中的所有内容

diff

比较文件，并列出他们的差异
diff filename1 filename2

chmod

更改文件的读取，写入和执行权限
chmod -options filename

gzip

压缩文件
gzip filename

gunzip

解压缩gzip压缩的文件
gunzip filename

gzcat

查看gzip压缩文件，而不需要gunzip它
gzcat filename

lpr

打印文件
lpr filename

lpq

查看打印机队列

lprm

从打印队列移除某些内容
lprm jobnumber

文本操作

awk

awk是处理文本文件最有用的命令
它一行一行地在整个文件上运行
默认情况下，它使用空格分隔字段

最常用的语法

-F指定在我们要基于哪个分隔字段

grep

查找文件内的文本
使用grep搜索与一个或多个正则表达式匹配的文本行
并仅输出匹配的行
grep pattern filename

-i强制忽略单词大小写
-r搜索指定目录下的所有文件
-w只搜索单词

wc

返回一个文件的行数，单词数，字符数
wc filename

sed

用于过滤和转换文本的流编辑器

用连字符替换所有空格

使用”d”替换所有的数字

sort

排序文本文件的行

随机化一个排序的example.txt

uniq

报告或省略重复的行

只显示example.txt的唯一行
(首先需要排序，否则看不到重叠)

显示每行的唯一项，并告诉找到了多少个实例

cut

从每行文件中删除部分

显示第2,7和9列，空格作为分隔符

echo

显示一行文字

用字母之间的换行显示 “Hello World”

fmt

简单的最佳文本格式化程序

将example.txt的行输出为20个字符的宽度

tr

翻译或删除字符

把所有小写字母变成为大写

把所有空格变成换行符

nl

显示文件的行数

带行号显示 example.txt

egrep

打印匹配模式的行

  • 扩展表达式
    别名：'grep -E'

显示“Lorem”或“dolor”的行

or

fgrep

打印匹配模式到的行

  • FIXED模式匹配
    别名：'grep -F'

找到具体的字符串’（Lorem | doloar）’

or

目录操作

mkdir

生成一个新的目录
mkdir dirname

cd

从一个目录转到另外一个目录

移动到主目录
$ cd
移动到该目录
cd dirname

pwd

返回目前所在的目录

SSH，系统信息，网络操作

ssh
