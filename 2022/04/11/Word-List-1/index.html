Word List 1

act——

act

vi.行动，起作用
v.扮演
n.表演，法令

1.act（行动）

active

a.【人或动物】活跃的，积极的，主动的
【语法】主动（语态）的【passive被动的】
【火山】随时会爆发的

  • actively
    ad.积极地，主动地
  • activist
    n.积极分子，活跃分子
  • activity
    n.活动
  • inactive
    a.不活跃的

activate

vt.【术语】使活动，激活

transact

v.做（交易），办理（业务）

transaction

n.交易，业务
（业务等的）办理，执行

act（起作用）

interact

vi.（with）
【事物】相互作用，相互影响
【人】交流，沟通，交往

interaction

n.相互作用，相互影响
交流

interactive

a.相互作用的，相互影响的
交互式的，人机对话的

  • interactive video
    交互式视频

react

vi.（to）（做出）反应
（to）（对食物等）有不良反应，过敏
（with）起化学反应

  • react against
    反抗，反对

reaction

n.反应
（药物或食物引起的）不良反应
化学反应
反动

reactionary

a.反动的
n.反动分子

act（扮演、法令）

enact

vt.扮演，演出
将…制定成法律

act【行动的结果 -> 事实（fact）】

actual

a.实际的，真实的
（the ~ sth.）真正的某事物

  • actually
    ad.实际上，事实上

actuality

n.（常 pl.）实际情况，真实情况
真实，实际

2.单词 act -> 词根 ag(it)，ag(on)(=act)

agency

n.代理处，经销处
（政府的）机构，局

agent
